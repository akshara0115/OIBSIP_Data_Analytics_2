# OIBSIP_Data_Analytics_2
# Customer Segmentation Analysis

## Project Overview
This project focuses on performing **customer segmentation analysis** for an e-commerce dataset. The goal is to group customers based on their purchasing behavior and identify distinct segments that can help businesses make data-driven decisions.

By applying **data analytics and machine learning techniques**, the project uncovers meaningful patterns in customer data and provides actionable business insights.

## Objectives
- Analyze customer purchasing behavior
- Segment customers into distinct groups using clustering
- Identify high-value and low-engagement customers
- Provide recommendations for targeted marketing strategies

## Dataset
The dataset used (`iFood.xlsx`) contains customer information such as:
- Demographics (age, income, etc.)
- Product spending (wine, fruits, meat, etc.)
- Purchase behavior (web/store purchases)
- Recency (last purchase)

## Technologies Used
- Python 
- Pandas & NumPy (Data Processing)
- Matplotlib (Visualization)
- Scikit-learn (Machine Learning - KMeans)
- Excel (Dataset)

## Project Workflow

1. **Data Collection**
   - Imported dataset from Excel file

2. **Data Cleaning**
   - Handled missing values
   - Ensured data consistency

3. **Feature Engineering**
   - Created:
     - `Total_Spending`
     - `Total_Purchases`

4. **Exploratory Data Analysis (EDA)**
   - Analyzed distributions and relationships

5. **Data Scaling**
   - Standardized features for clustering

6. **Customer Segmentation**
   - Applied **K-Means Clustering**
   - Used **Elbow Method** to determine optimal clusters

7. **Visualization**
   - Scatter plots for cluster representation

8. **Insights & Recommendations**
   - Interpreted customer segments for business use

## Key Features
- End-to-end data analytics pipeline
- Customer behavior analysis
- Unsupervised machine learning (K-Means)
- Data visualization
- Business insight generation

## Results & Insights
- Identified different customer segments such as:
  - High-value customers
  - Frequent buyers
  - Low-engagement customers
- Enabled targeted marketing strategies
- Improved understanding of customer behavior

## Business Recommendations
- Offer loyalty rewards to high-value customers
- Provide personalized discounts to inactive users
- Target frequent buyers with upselling strategies
- Re-engage low-frequency customers through campaigns
